# 专注力波浪计时器

一款视觉化的专注力和休息时间管理工具，结合番茄工作法和动态波浪动画，帮助你提高工作效率和注意力管理能力。

## 功能特性

### 核心功能
- **专注/休息计时**：可自定义专注时间和休息时间，自动切换模式
- **动态波浪动画**：实时的波浪动画反映剩余时间，视觉化工作状态
- **注意力波动模拟**：模拟真实的注意力波动曲线，在专注模式下提供动态反馈
- **波浪提醒**：定时提醒用户调整注意力，避免注意力分散

### 视觉体验
- **渐变背景**：平滑的渐变背景动画，创造沉浸式专注环境
- **粒子效果**：动态背景粒子增强视觉体验
- **波纹交互**：点击波浪区域产生互动波纹效果
- **完成动画**：任务完成时显示庆祝动画和提示

### 用户友好设计
- **实时进度条**：直观显示当前计时进度
- **响应式布局**：完美适配桌面和移动设备
- **自定义设置**：灵活配置专注时间、休息时间和提醒频率
- **声音提示**：模式切换时提供音效反馈

### 移动设备优化
- **红米14pro完美适配**：针对大屏智能手机优化的触摸体验和视觉效果
- **触摸事件支持**：优化的触摸交互，防止触摸冲突和误操作
- **防抖处理**：优化输入和窗口调整性能
- **触摸反馈**：按钮点击反馈和视觉响应

## 技术栈

- **HTML5**：语义化标记和Canvas元素
- **CSS3**：现代CSS特性，包括渐变、动画、变量和响应式设计
- **JavaScript**：核心功能实现，包括计时器、动画和交互逻辑
- **Canvas API**：波浪动画和粒子系统的绘制

## 界面介绍

### 主界面组件
- **计时器显示**：中央显示当前剩余时间
- **会话类型**：显示当前是「专注时间」还是「休息时间」
- **波浪区域**：动态显示剩余时间的波浪动画
- **控制按钮**：开始、暂停、重置计时器
- **设置区域**：调整专注时间、休息时间和提醒频率
- **信息区域**：应用介绍和使用提示

### 自定义选项
- **专注时间**：默认为25分钟，可自定义
- **休息时间**：默认为5分钟，可自定义
- **波浪提醒**：默认为10分钟，可自定义提醒频率

## 使用建议

1. **设定合理时间**：根据个人工作习惯调整专注和休息时间
2. **专注时避免干扰**：关闭通知，创造安静环境
3. **充分休息**：休息时间完全放松，避免查看工作相关内容
4. **定期调整**：根据实际使用效果调整提醒频率
